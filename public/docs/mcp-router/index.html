Routing Contract

  • Intelligence traffic must use /api/v1/intelligence/*.
  • Do not call browser-side Edge Function routes such as /functions/v1/intelligence-*.
  • MCP Services and API Keys UI are wired to live managers and production-backed data flows.

Dashboard API Base URL

API base resolution in src/lib/api-client.ts:

  1. VITE_CORE_API_BASE_URL (preferred)
  2. VITE_API_URL fallback (legacy; trailing /v1 stripped)
  3. https://api.lanonasis.com default

Set origins only. Do not append /api/v1 and avoid a trailing /v1 on VITE_CORE_API_BASE_URL.
